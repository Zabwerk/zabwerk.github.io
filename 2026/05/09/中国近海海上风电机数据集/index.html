🌊中国近海海上风电机数据集 (2015-2025)

China Offshore Wind Turbines Dataset (COWT, 2015-2025)

本数据集基于Google Earth Engine (GEE)平台，利用多时相Sentinel-1 SAR影像提取，系统性地呈现了中国近海海域风电机组在2015年至2025年间的时空演变格局。

📊 数据集摘要

  • 覆盖范围： 中国近海海域
  • 时间跨度： 2015年-2025年
  • 数据源： Sentinel-1 SAR(Google Earth Engine)
  • 空间分辨率： 约10米(Sentinel-1 IW)
  • 数据格式： Vector(Shapefile)
  • 总体精度： 98%

📂 数据内容与结构

本仓库主要包含最终处理的 Shapefile 数据文件。数据以单个点要素（风电机组）的形式存储。
主要属性字段：
| 字段名 (Field) | 描述 (Description) |
| :--- | :--- |
| lon | 风电机组中心点经度 |
| lat | 风电机组中心点纬度 |
| time | 建设时间 (精确到日/月) |
| year | 建设年份 |
| name | 所属风场名称 |

注： 坐标系建议使用WGS84(EPSG:4326)。
