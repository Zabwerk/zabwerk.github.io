化学原创题-1

发表于2026-02-14|更新于2026-02-14|化学初中化学
|浏览量:

Velpatasvir（\(C_{48}H_{51}O_8N_8\)）是一种弱碱性抗病毒药物，用于治疗丙型肝炎，是一种质子泵抑制剂，用于治疗胃溃疡、十二指肠溃疡、应激性溃疡反流性食管炎等疾病。荷兰Radboud大学医学中心的Minouvan Seyen等人近期发表在Clinical Pharmacology & Therapeutice杂志上的文章，研究了可口可乐对于该药物的影响。研究表明临床上有很多丙肝病人同时患有胃病，在服用Velpatasvir时还需要服用奥美拉唑，两种药物存在相互作用，导致Velpatasvir的血药浓度下降，而此时服用可口可乐，能够将其血药浓度显著提高，原因是可口可乐的酸性（可乐中含有易分解的碳酸和不易分解的磷酸）使得Velpatasvir的溶解度增大，使更多的药物被人体吸收。下列说法正确的是__________。

请选择：

A

Velpatasvir中含有四种元素均属于第二周期的元素

0 票
B

1分子Velpatasvir的碳、氢、氧三种元素的质量之比为48:51:8

0 票
C

患者服用可口可乐后，可乐中的碳酸因收到人体温度分解为$H_2O$和$CO_2$,属于分解反应。

0 票
D

可口可乐的pH<7，是一种酸。

0 票

正确答案：C

解析：碳酸分解的化学方程式为 $\ce{H2CO3 = H2O + CO2 ^}$，这是一个分解反应。

注意：

  • A选项错误，$H$属于第一周期元素
  • B选项错误，质量比应该是 $(48×12):(51×1):(8×16) = 576:51:128$，而$48:51:8$表示的是一分子的碳、氢、氧原子数之比。
  • D选项错误，可口可乐是混合物，$pH<7$只能说明该溶液呈酸性，但不一定是酸。
